# Windows

Change Windows: `Alt` + `Tab` (or `Win` + `Tab`)

Cut: `Ctrl` + `X`
Copy:: `Ctrl` + `C`
Paste: `Ctrl` + `V`

# Web Browsers

Next Tab: `Ctrl` + `Tab`
Prev Tab: `Shift` + `Ctrl` + `Tab`

# Windows Terminal

Open the Command Palette with `CTRL` + `SHIFT` + `P`

Then start typing one of these:

```
Move tab backward
Move tab forward
```
