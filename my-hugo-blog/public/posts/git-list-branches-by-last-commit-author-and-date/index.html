List git branches with the date and author of the last commit

Jul 24, 2023 
git branch --all --sort=creatordate --sort=committername `
  --format "%(align:20) %(creatordate:short) %(end) %(align:25) %(committername) %(end) %(refname:lstrip=-1)"
